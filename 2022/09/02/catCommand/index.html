cat命令的功能是把文件的内容显示到标准输出，用法格式如下：

短选项 长选项 含义
--help 打印帮助信息
--version 打印版本号
-n --number 给输出的行编号，从1开始

-s --squeeze-blank 合并相邻的多个空行为一个空行
-v --show-nonprinting 展示除了LFD（换行）和TAB（水平制表符）以外的所有控制字符，展示方式是使用^标记，例如ASCII码中的NUL字符可以使用CTRL+@表示，那么NUL字符就展示为^@。
-v选项使得cat命令一个字节一个字节的展示文件的数据，cat会默认把一个字节的低7位当作ASCII码来展示，如果是可打印字符就展示对应的字符，如果不是可打印的，就使用^+xxx来展示，例如当这个字节是0的时候，就是展示成^@，如果这个字节的最高位是1，那么就在展示的字符前面加上前导的M-，例如-128，就展示为M-^@
-E --show-ends 把换行符LFD展示为$，如果在换行符前面有回车符，即\r\n，则一起展示为^M$
-T --show-tabs 把TAB符展示为^I（爱）
-e 等于-vE
-t 等于-vT
-A 等于-vET

3. nl命令

格式

nl命令把每一个文件的内容写到到标准输出，写出的时候给文件内容加上行号。

nl命令把所有的文件的内容依次连接在一起，看作一个完整的文档，然后根据分页标记把整个文档分成若干页，在文件之间分页和行号不会被重置。

逻辑页由头，主体和页脚组成，每一个部分的内容都可以以不同的方式对每一行进行编号。

页的每一部分都有一个分割符，页头的分割符是“:::”，页主体的分割符是“::”，页脚的分割符是“:”，这些分割符可以修改，但是其格式不能被修改。

输出的时候，页区分隔行会被替换成空行。第一个分区之前的内容会被认为是主体部分，因此如果一个文件中没有任何一个分区，那么所有的内容会被看作主体部分。

-b style

--body-numbering=style

这个选项指定页主体部分的行号添加的格策略，style有如下可选的值：

  • a：为所有的行添加行号
  • b：只为非空行添加行号，页主体部分默认是这个
  • n：不添加行号，页头和页脚默认是这个格式

-d cd

--section-delimiter=cd

这个选项设置页区的分割符，cd是两个字符，如果只有c一个字符，那么第二个默认是“:”字符，作为GNU扩展名，可以指定两个以上字符，并且如果cd是空串，你那么就会禁用页分区功能。

-f style

--footer-numbering=style

页脚部分的行号格式

页头部分的行号策略，和-b选项类似的。

-i number

--line-increment=number

行号的增量，默认是1，可以指定负数。

-l number

--join-blank-lines=number

把number个连续的空行看作一个空行，并且只为最后一行设置行号（这些空行都会被打印出来的），当少于number个空行的时候，不为他们设置行号。空行指的是除了换行字符，没有任何字符的行。

4. od命令

od命令支持把文件内容以字节块的形式查看，默认情况下，od展示的每一行的内容格式如下：

offset byte_block1 byte_block2 byte_bloc3 byte_block4……

offset使用8进制数字表示，一个byte_block是C语言中 short int的长度，一般是2字节，用8进制数字表示。例如一个文件中的内容只有一个字节，且这个字节的值是0

那么使用od命令显示的结果是：

-A radix

--address-radix=radix

这个选项指定了offset的显示格式,radix的取值如下：

  • d：十进制
  • o：八进制（默认选择）
  • x：十六进制
  • n：不显示offset

-j bytes
--skip-bytes=bytes

这个选项表示跳过bytes个字节。如果bytes以0x或0X开头，那么bytes是十六进制的数字，如果以0开头，那么是8进制的数字，否则bytes被认为是十进制的。
bytes后面可以跟单位后缀，即：

  • ‘b’ => 512 (“blocks”)
  • ‘KB’ => 1000 (KiloBytes)
  • ‘K’ => 1024 (KibiBytes)
  • ‘MB’ => 1000*1000 (MegaBytes)
  • ‘M’ => 1024*1024 (MebiBytes)
  • ‘GB’ => 100010001000 (GigaBytes)
  • ‘G’ => 102410241024 (GibiBytes)

-t type

--format=type

指定数据展示的格式，type取值如下：

  • a：一个字节是一组，且只看低7位组成的ASCII的值，如果是可打印的字符，那么打印其ASCII值，如果是空格，打印sp，如果是换行打印nl，是0字节则打印nul
  • c：一个字节一组，如果是可打印的字符，那么打印其ASCII的值，否则如果可以使用反斜杠转义的C语言风格的字符，则转义后打印，否则使用3位8进制数字表示这个字节。
  • d：有符号十进制整数，4个字节一组。
  • f：浮点数，8个字节一组
  • o：8进制整数，4字节一组
  • u：无符号十进制整数，4字节一组。
  • x：十六进制，4个字节一组。

  • -x，等于 -t x2

--endian=order

这个选项指定了多字节数据按照什么字节顺序展示，order有little（小端方式）和big（大端方式）。

默认按照系统字节序展示。

-w[n]

--width[=n]

这个选项指定了od命令一行显示几个字节，默认16字节，如果n省略，就是32字节。
